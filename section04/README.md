<h3>React 컴포넌트의 라이프사이클 제어(LifeCycle Control)</h3>

# Mount -> Update -> Unmount

# 1. Mount = 탄생 : 컴포넌트가 화면에 렌더링 됨
# 서버에서 데이터를 불러옴

# 2. Update = 변화 : 컴포넌트가 리렌더 됨
# 어떤 값이 변경되었는지 콘솔에 출력

# 3. Unmount = 죽음 : 컴포넌트가 화면에서 제거 됨
# 컴포넌트가 사용하던 메모리 정리 및 해제

# useEffect를 이용하면 쉽게 할 수 있음
